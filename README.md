### How to run

* Split a new terminal and run this:

* Install app using npm:

`npm install`

* Or using yarn:

`yarn install`

* Run app using npm:

`npm run dev`

* Or using yarn:

`yarn dev`

* Open: 

[http://localhost:5173/](http://localhost:5173/)
